多级目录菜单
===
* menu.py
    描述信息：
        入口程序,用于查看公司信息，可根据城市，区进行查找
        该程序仅适用于 python3.x，如 python 版本为 2.x，请升级至 3.x
    执行方法：
        1、chmod +x login.py; ./menu.py
        2、python menu.py

* setting.py
    描述信息：
        城市，区，公司的信息信息列表，作为一个模块导入到 menu.py 中

* menu-flowchart.png
    描述信息：
        menu.py 程序的处理逻辑图
