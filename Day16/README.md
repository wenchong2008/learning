##主要功能

* 主页可跳转至登陆、注册、购物车页面
* 登陆页面:
    1、点击登陆时先验证用户输入，并增加错误提示
    2、跳转至注册页面
* 注册页面：
    1、点击注册时先验证用户输入，并增加错误提示
    2、当用户点击输入框是，会出现该输入框的提示信息
* 购物车页面：
    1、可选择输入商品数量，并自动计算价格
    2、全选或取消商品，自动计算价格
    3、删除商品，自动计算价格
    4、单选商品，自动计算价格
