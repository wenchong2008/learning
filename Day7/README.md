                python select_course.py       # 程序运行
                
    2、config: 程序配置文件目录
            setting.py: 程序配置文件
            
    3、database: 程序数据信息存放目录
            
    4、lib: 程序公共文件目录

    5、src: 程序逻辑函数目录
            GameChangeLife.py: 改变人生模块
            GameUserAdmin.py: 用户管理模块
            VirtualLife.py: 主模块
            
    6、VirtualLife.png: 流程图
